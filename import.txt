Етапи імпорту / запуску файлу в ноді:
1. resolving - етап знаходення файлу
Нода вважає, що:
  1.1 імпортується модуль:
    1.1.1 нода шукає вашу назву у своїх модулях
    1.1.2 нода шукає вашу назву у node_modules
  1.2 імпортується файл:
    1.2.1 нода шукає файл з розширенням .js
    1.2.2 нода шукає файл з розширенням .json
  1.3 імпортується папка:
    1.3.1 всередині буде шукатися package.json з властивістю "main"
    1.3.2 всередині буде шукатися файл index.js
    1.3.3 всередині буде шукатися файл index.json
  1.4 кидається помилка
2. loading - ваш знайдений файл нода завантажує
3. wrapping - код з завантаженого файлу з минулого кроку нода обготрає у функцію, в яку докидує
додаткові injected дані 
4. evaluation - функцію з попереднього етапу запускають 
(для одного файлу цей етап виконується тільки 1 раз)
5. caching - результати роботи evaluation зберігаються і віддються під час
кожного наступного імпорту
